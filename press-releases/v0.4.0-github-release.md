# v0.4.0 вҖ” Task Tray GUI, Shared DB Module, Utility Scripts & Bug Fixes

**sqlite-memory-mcp now ships a native system tray task manager (PyQt6), a shared `db_utils` module eliminating cross-file duplication, five standalone utility scripts, and 15 bug fixes.**

Tool count: 21 (unchanged). Total codebase: ~2,460 lines across 8 Python files.

---

## What's new

### Task Tray вҖ” PyQt6 System Tray App

A native desktop task manager that reads/writes directly to `memory.db`:

- **System tray icon** with overdue badge counter (red circle, 9+ overflow)
- **Compact popup** вҖ” Today + Overdue tasks, checkbox toggle, quick-add input
- **Full window** вҖ” tabbed view (Today / Inbox / Next / All), double-click to edit, right-click to delete
- **Auto-refresh** every 30 seconds when visible
- **Window geometry persistence** via QSettings

```bash
python3 task_tray.py
```

Runs as a tray icon вҖ” left-click toggles popup, right-click shows context menu with "Open Full Window", "Add Task", "Quit".

### Shared DB Module вҖ” `db_utils.py`

Single source of truth for constants and helpers used across all files:

```python
from db_utils import (
    DB_PATH, BRIDGE_REPO,
    TASK_SECTIONS, TASK_PRIORITIES, TASK_STATUSES,
    TASK_HIDDEN_STATUSES, PRIORITY_RANK, PRIORITY_COLORS,
    TASK_ALLOWED_UPDATE_FIELDS,
    get_conn,          # context manager with WAL + foreign_keys + busy_timeout
    now_iso,           # UTC ISO 8601 timestamp
    parse_iso_date,    # safe YYYY-MM-DD -> date | None
    is_overdue,        # consistent overdue check
    build_priority_order_sql,  # CASE clause for SQL ORDER BY
    priority_sort_key,         # Python sort key (rank, due)
)
```

Eliminates ~93 lines of duplicated DB connection setup, constant definitions, and timestamp helpers across 7 files.

### Utility Scripts

Five standalone scripts for task automation (all use `db_utils`):

| Script | Purpose |
|--------|---------|
| `auto_archive.py` | Archive done tasks older than N days (`--days 7 --dry-run`) |
| `daily_digest.py` | Print markdown task digest to stdout (`--sections today,inbox,next`) |
| `overdue_bump.py` | Bump priority of overdue tasks (`--target high --dry-run`) |
| `recurring_tasks.py` | Recreate recurring tasks based on JSON schedule (`--dry-run`) |
| `task_report.py` | Generate static HTML Kanban board for GitHub Pages |

All support `--db PATH` to override the default database location.

### Bug Fixes

**v0.3.0 code review (13 bugs):**
- Fixed in commit `33b2e26` вҖ” see changelog

**Task Tray quality fixes:**
- Explicit dark text color in FullWindow for contrast on light backgrounds
- Spec compliance: menu structure, QSettings, screen centering
- Edge case tests for callbacks and data layer

**db_utils refactoring fixes:**
- `bridge_push()`: git writes "nothing to commit" to stdout, not stderr вҖ” early-return path was dead code
- `TASK_ALLOWED_UPDATE_FIELDS` was missing `parent_id` вҖ” tray's `update_task()` would reject subtask reassignment
- Fragile overdue check (string comparison) replaced with `is_overdue()` using proper date parsing
- Priority colors dict moved out of hot loop in tray popup rendering
- Double `get_summary()` call in tray refresh eliminated (was querying DB twice per update)
